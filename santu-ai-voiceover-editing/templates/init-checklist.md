# 黑底头像声波口播初始化清单

复制本清单到新项目或需求文档里，开工前逐项确认。

## 1. 账号与声音

- [ ] 使用默认三土音色 `SantuVoice20260618B`。
- [ ] 或改用自有 MiniMax 克隆音色：`voice_id = __________`。
- [ ] 或已有成品音频，跳过 TTS：`audioPath = __________`。
- [ ] MiniMax API key 已在本机环境配置，且没有提交到仓库。
- [ ] 如果要克隆新音色，源音频干净、无背景音乐、无多人说话。

## 2. 头像与人物形象

- [ ] 使用默认头像：`assets/santu-avatar.png`。
- [ ] 或提供新头像：`avatarPath = __________`。
- [ ] 新头像已裁成正方形或透明圆形 PNG。
- [ ] 插画固定人物使用默认“戴眼镜西装小人”。
- [ ] 或自定义插画人物 IP：`__________`。

## 3. 文案与封面

- [ ] 已提供中文口播文案。
- [ ] 文案中不适合朗读的括号、列表符号、视觉说明已清理。
- [ ] 是否生成封面：是 / 否。
- [ ] 封面标题：`__________`。
- [ ] 需要 `9:16` 和 `6:7` 两版封面，除非用户明确只要一种。

## 4. 插画资产

- [ ] 每 `4-7 秒` 一个视觉单元。
- [ ] 每张图只表达一个判断、流程、状态或隐喻。
- [ ] prompt 已要求黑底、白/灰线稿、少量蓝橙红。
- [ ] prompt 已要求主体在上方视觉区，并预留字幕/头像/声波区域。
- [ ] 图片已从 `$CODEX_HOME/generated_images/...` 复制到项目 `public/media/images/`。
- [ ] `asset-manifest.json` 已记录真实像素尺寸。

## 5. Remotion 与布局

- [ ] 画幅 `1080x1920`，背景 `#050505`。
- [ ] 插画按真实尺寸等比例显示，不拉伸、不裁切。
- [ ] 左右平台安全边距至少 `80px`，顶部至少 `60px`。
- [ ] 字幕区在头像上方，最多 2 行。
- [ ] 字幕黑影安全带覆盖字幕区。
- [ ] 头像圆心约 `X=540, Y=1400`，头像左右声波不压字幕。

## 6. 交付检查

- [ ] `npm run validate-layout` 通过。
- [ ] `npm run typecheck` 通过。
- [ ] 最终 MP4 已写入 `out/`。
- [ ] `ffprobe` 确认 `1080x1920`、`30fps`。
- [ ] 已从最终 MP4 抽关键帧检查，不只看预览。
- [ ] 封面标题人工检查无错字、漏字、多字或乱码。
