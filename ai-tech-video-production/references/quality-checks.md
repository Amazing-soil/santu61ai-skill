# Quality Checks

## Layout

- [ ] 非图片素材（React/Remotion/Anime.js、文字信息图、流程卡、图表、代码/窗口模拟）没有贴近屏幕上沿。
- [ ] `Y=0-260` 只放背景氛围；首个核心文字/卡片/图标组/流程节点顶部不进入顶部安全区，建议 `Y>=280`。
- [ ] 非图片素材核心可读信息主要位于 `Y=280-1080`。
- [ ] 对比/反常识段使用分屏对照；步骤/特征/判断标准段使用 HUD 坐标清单，未回退到发光小矩形框标签。
- [ ] 烧录字幕位于中间偏下安全区，而不是贴正下方。
- [ ] 正文阶段字幕不遮挡中间下方真人 PIP，PIP 也不压字幕；开头/结尾居中强化出镜时字幕不压人物。
- [ ] 字幕位于不挡住人像的位置：正文不进入中间下方 PIP 区；开头/结尾不压脸、嘴、手和胸口。
- [ ] 核心文字和关键图形不进入中间偏下字幕区。
- [ ] 正文阶段中间下方真人 PIP 没遮住主素材。
- [ ] 如果不烧录字幕，中间偏下区域可给后期字幕使用。
- [ ] PIP 人像没有被拉伸。
- [ ] PIP 是原口播视频，不是素材。
- [ ] 开头前 `3 秒` 人物居中放大但没有全屏，背景素材和钩子辅助信息仍然成立。
- [ ] 最后 `3 秒` 人物回到中央，背景素材和结论/提问辅助信息仍然成立。
- [ ] 居中人物没有遮挡主标题、关键标签或字幕，人物脸部与胸口没有被文字覆盖。

## Timing

- [ ] 素材出现时间与口播语义对齐；关键素材不晚于对应核心词后 `0.3 秒` 出现。
- [ ] 开头钩子、数字/结论句、转折句和 CTA 已抽查音频时间点，没有按脚本文案硬配。
- [ ] 如果生成字幕，字幕时间和文字以真实语音口播为准；用户文案/旧字幕只做参考，没有被直接照搬。
- [ ] 字幕按自然句、语义短句或停顿切分；每句尽量单行，未被机械拆成多行碎片。
- [ ] 每 `30 秒` 至少有 `1` 个 GPT Image / AI 图片素材，最好 `1-2` 个；全实操录屏窗口可例外。
- [ ] 太短段落没有硬塞播不完的素材。
- [ ] MP4 动效已按段落时长变速。
- [ ] 每 `3-5 秒` 有视觉变化，但不为变化而变化。

## Transition

- [ ] 开头人物从居中强化出镜平滑缩小并移动到中间下方，没有硬切或跳位。
- [ ] 结尾人物从中间下方平滑回到中央，没有硬切或跳位。
- [ ] 相邻素材没有露出口播底图。
- [ ] 桥接空档不会闪屏。
- [ ] 覆盖式淡入时上一段素材保持不透明。

## Content

- [ ] 字幕轨可编辑，字幕文字、时间、位置和样式在 `src/timeline.json` 或等价配置中可改。
- [ ] 字幕组件继承全局 CSS 的 PingFang SC 字体栈；Mac 渲染优先苹方，字重为 `900`。
- [ ] 字幕为白字、黑色阴影、动态字号；长句优先缩小字号或语义拆条，没有挤出容器。
- [ ] 没有未核验新闻截图或虚假数据。
- [ ] 实操段没有被概念图盖住。
- [ ] 不是整条视频都用文字信息图；已混合 GPT Image/AI 图片、React/Remotion/Anime.js 动效、实操/截图等素材类型。
- [ ] 没有真实实操录屏时，成片至少包含 `1` 个 AI 图片段和 `1` 个 React/Remotion/Anime.js 动效段。
- [ ] 短标签没有按钮感、表单感、后台面板感；没有绿色/红色描边矩形框包短句。
- [ ] GPT Image 图片不包含字幕、标题、金句、品牌 Logo、真实 UI 或假冒截图。
- [ ] 文字短句化，适合中文观众快速读懂。
- [ ] 术语有必要时用用户语言解释。

## Delivery

- [ ] 有最终 MP4。
- [ ] 最终 MP4 是高码率交付版，不是低码率预览版；默认命名包含 `final_hq` 或在交付说明中明确为高码率最终版。
- [ ] `ffprobe` 已验证最终 MP4：`1080x1920/30fps/H.264` 默认视频码率约 `8M` 或更高；复杂动效/录屏较多时约 `10M-12M`。
- [ ] 如果另有小文件版本，它只作为 `preview/low` 预览版，不能替代高码率最终版。
- [ ] 干净交付区只保留最终视频、实际使用/生成的图片素材、预览检查图，除非用户明确要求保留工程。
- [ ] 有关键帧检查图、封面预览或 contact sheet。
- [ ] 有素材目录说明，且只包含实际使用或需要保留的图片素材。
- [ ] 自动清理只处理当前任务新建的临时工程和中间文件，没有删除原始视频、已有素材目录或其他任务工程。
- [ ] 如果用户要求可编辑工程，有快速修改入口，例如 `src/timeline.json`。
