# 剪辑师交付说明

## 基本规则

- 画幅：1080x1920。
- 默认不额外加字幕，保留中间偏下字幕区。
- 真人 PIP 默认右下 `300x400`，圆角，等比例裁切。
- 非图片素材整体下移，`Y=0-260` 只放背景氛围；核心文字、卡片、图标组、流程节点优先放在 `Y=280-1080`。
- 素材核心信息集中在安全的中部偏上区域，不贴屏幕上沿，不进入中间偏下字幕区和右下 PIP 区。
- 相邻素材转场使用覆盖式淡入，不能闪回口播底图。

## 可修改项

- 删除素材：禁用对应段落。
- 换素材：替换素材路径。
- 调时间：修改 start / end。
- 调 PIP：修改 x / y / width / height / radius。
